API Access

Use this key to authenticate with the JSON API

{{.APIKey}}

This key is generated when the server starts and remains valid until the server is restarted.

Usage Examples

List VMs

curl -H "Authorization: Bearer {{.APIKey}}" \
  http://{{.ListenAddr}}/api/v1/vms

Create a VM

curl -X POST -H "Authorization: Bearer {{.APIKey}}" \
  -H "Content-Type: application/json" \
  -d '{"name":"my-vm","cpus":2,"memory_mb":1024}' \
  http://{{.ListenAddr}}/api/v1/vms

Start a VM

curl -X POST -H "Authorization: Bearer {{.APIKey}}" \
  http://{{.ListenAddr}}/api/v1/vms/my-vm/start

Health Check (no auth required)

curl http://{{.ListenAddr}}/api/v1/health
